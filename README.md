# Sample — Restaurant Order Management System

Personal research sample. A waiter-facing tablet/mobile app for restaurant order capture, with table allocation and kitchen submission.

See [SPEC.md](SPEC.md).

## What this sample shows
- Hardware/software constraint framing (Android/iOS, tablet form factor)
- Multi-actor role model (Shift Manager / Admin · Waiter · Kitchen)
- Notification-driven workflow (table allocation pushed to waiter)
- Wireframe-level UX flow narration with pre-conditions
- Pragmatic scope discipline ("Detailed flow can be improved based on priority from stakeholders")
